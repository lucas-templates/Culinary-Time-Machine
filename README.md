## üìã Project Overview

**Culinary Time Machine** is a stunning single-page landing website that takes users on a gastronomic journey through time. The project showcases a fictional meal kit service offering historically accurate recipes from different eras including the Viking Age, Ancient Rome, Victorian England, and 1920s America.

This interactive website features:
- **Dynamic era selection** with smooth transitions
- **Interactive timeline** with hover effects and animations
- **Artifact unpacking animation** that reveals meal kit contents
- **Responsive design** optimized for all devices
- **Rich visual aesthetics** with period-appropriate color schemes

The project demonstrates modern web development techniques including CSS animations, JavaScript interactivity, and responsive design principles.

## üöÄ Setup Instructions

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, but recommended for best experience)

### Option 1: Direct File Opening
1. **Download or clone** the project files
2. **Navigate** to the project directory
3. **Double-click** `index.html` to open in your default browser

### Option 2: Local Web Server (Recommended)

#### Using Python
```bash
# Navigate to project directory
cd "Culinary Time Machine"

# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```
Then open `http://localhost:8000` in your browser.

#### Using Node.js (http-server)
```bash
# Install http-server globally (one-time)
npm install -g http-server

# Navigate to project directory
cd "Culinary Time Machine"

# Start server
http-server -p 8000
```
Then open `http://localhost:8000` in your browser.

#### Using VS Code Live Server
1. Install the **Live Server** extension in VS Code
2. Right-click on `index.html`
3. Select **"Open with Live Server"**

### Option 3: Deploy to Web Hosting
Upload all project files to any web hosting service:
- GitHub Pages
- Netlify
- Vercel
- Traditional web hosting (cPanel, etc.)
